Schneider-Eletric

Software Engineering Intern | C#/.NET, Azure

At Schneider-Electric, I worked in Asset Development for HMI (Human Machine Interface) Software Solutions. Within this, I worked in back-end development, specifically with databases & server-side programming.

One of the larger functions I provided was my extension of a time-series cloud database's ability to showcase various different data patterns by developing and deploying a customizable Azure Cloud Service Worker Role that programmatically queries online for stock data and records them in the database. I wrote the functionality in C# and queried Yahoo Finance using YQL (Yahoo Query Language).

I also implemented a diagnostics paradigm in C# for a time-series cloud database. This protocol would allow the database to observe and record its own internal communications, server calls, and general functionality to self-diagnose in case of crashes or errors. This work involved technical details such as programmatic access of non-relational Azure storage accounts, parallelized network calls via TPL (Task Parallel Library) to a series of APIs, and large volume JSON object creation and management.
